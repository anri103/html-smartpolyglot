# Сайт Smart Polyglot

Проект представляет собой компонентную вёрстку многостраничного сайта для обучения иностранным языкам.

## Используемые технологии

- HTML5
- CSS3
- SCSS
- Vanilla JavaScript
- Bootstrap 5
- jQuery
- Gulp

## Ссылки

- [GitHub Pages](https://anri103.github.io/html-smartpolyglot/dist/index.html) — Вёрстка проекта.

## Разработка

1. Установить зависимости:

```bash
npm i
```

2. Запустить разработку:

```bash
npm run dev
```

3. Сборка проекта:

```bash
npm run build
```

## Итоговая вёрстка

Все файлы итоговой вёрстки находятся в директории `dist/`.
